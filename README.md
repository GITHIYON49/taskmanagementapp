TaskManager - Full Stack Task Management Application



## Features

### Authentication & Authorization
- User registration and login with JWT authentication
- Password encryption with bcrypt
- Persistent login sessions

###  Project Management
- Create, read, update, and delete projects
- Project status tracking (Planning, Active, On Hold, Completed, Cancelled)
- Priority levels (Low, Medium, High)
- Progress tracking with visual indicators
- Project analytics and statistics
- Project calendar view
- Team member management per project

### Task Management
- Create and assign tasks to team members
- Task types (Task, Bug, Feature, Improvement, Other)
- Task status workflow (To Do, In Progress, Completed)
- Priority levels for tasks
- Due date tracking with reminders
- Task descriptions and comments
- File attachments (up to 5MB per file)
- Task sharing with permission levels (View/Edit)

### Team Collaboration
- Invite team members via email
- User roles (Admin/Member)
- Project member management
- Task assignment notifications
- Email notifications for:
  - Task assignments
  - Task sharing
  - Due date reminders (24 hours before)
  - Member invitations
- Real-time collaboration features

### Notifications
- In-app notification center
- Unread notification badges
- Email notifications for important events
- Automated daily reminders at 9:00 AM
- Mark as read/unread functionality

### Responsive Design
- Mobile-first responsive design
- Works on all devices (mobile, tablet, desktop)
- Touch-friendly interface
- Collapsible mobile sidebar
- Adaptive layouts and components

### Analytics & Reporting
- Project analytics dashboard
- Task completion statistics
- Team performance metrics
- Progress visualization with charts
- Activity timeline

### User Experience
- Clean, modern UI with Tailwind CSS
- Intuitive navigation
- Real-time updates with Redux
- Toast notifications for user feedback
- Loading states and error handling
- Dark mode support (settings page)
